About Git

At its core, Git is a set of command line utility programs that are designed to execute on a Unix style command-line environment. Modern operating systems like Linux and macOS both include built-in Unix command line terminals.

In Windows environments, Git is often packaged as part of higher level GUI applications. GUIs for Git may attempt to abstract and hide the underlying version control system primitives. This can be a great aid for Git beginners to rapidly contribute to a project.

My Favourite Git Command

CD happens to be my favourite git command, the first command i learnt in git command line (Git Bash), CD simply means Change Directory


Function of CD:

"CD" helps user to change current working directory or switch into a Directory (folder), which really is a great deal when it comes to do with formating of directories

CD command can be used with: ".." or "~" flags.

Exammple:

With (CD .. ) you move a step back from a current working directory to immediate following directory, also

With (CD ~) you move from a current working directory to a home directory, which is the begining of your directories

Simple example of how this commands work
